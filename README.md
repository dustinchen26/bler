## How to use
```
Input the du_stats_XXX.txt, then it will parse the ( DL-SUCC, DL-NEWTX, DL-RETX, BLER, SCH  DL Tpt, UL Tpt )  

Step:
1.	decompress the BlerTputParser.7z
2.	Put any type of du_stats_XXX log in the decompressed folder
3.	Execute the BlerTputParser.exe
4.	View the calculation results parse_DL_BLER_results.txt

Calculation formula:
BLER = DL-RETX / (DL-NEWTX +DL-RETX) 
```
